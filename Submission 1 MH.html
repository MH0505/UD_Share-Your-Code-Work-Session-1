Intro to the Web

The Web is essentially a collection of HTML files that are linked together via so called Hyperlinks. An HTML file ("Hypertext Markup Language") is the main document type on the web, comparable to MS office products on Microsoft.

The World Wide Web: Components

The key components of the Web as displayed in the picture set forth below are
  • the browser on my computer, which is the gateway to the internet; i.e. a program displaying files on the web;
  • the internet, which is a large computer network;
  • the HTTP protocol ("Hypertext Transport Protocol"), i.e. a set of rules managing the communication between the browser and the server to display the appropriate HTML files;
  • the server, which is a computer hosting the HTML files that make up the web

Notes on HTML

HTML is made up of
  • text content "the plain text that one will ultimately see"; ""
  • markup language "tags that will style text and video content ";
  • references to videos and pictures;
  • links to other webpages;

Markup Language, i.e. "tags"

Markup language consists of tags that alter the appearance of what is "seen" on the actual website. Tags consist of opening tags and closing tags with content in between, together called "element"

Set forth below are various tags that were part of the course. Further tags can be found on websites such as W3Schools.com
  • The tag "b" writes contents in bold and is an inline element
  • The tag "em" writes contents in italics and is an inline element
  • The tag "br" inserts a line break and is an inline element
  • The tag "a href " references to other websites, i.e. defines a hyperlink and is an inline element
  • The tag "img" inserts an image by using a link and is an inline element
  • The tag "p" is a paragraph and is a block element, i.e. puts a box around the contents inside the tag
  • The element "span" is an inline element and needed to "style with CSS"
  • The element "div" is a block element and needed to "style with CSS"

Basics on HTML document structures

HTML documents are structure into the following parts
  • Doctype definition, which is typically HTML 5
  • An HTML tag / element
  • a header tag containing title, CSS information and other metadata
  • the body tag, which is the main focus of this course
